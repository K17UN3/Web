【Dockerのインストール】
１．sudo yum install -y docker

２．sudo systemctl start docker

３．sudo usermod -a -G docker ec2-user


【Docker Composeのインストール】
１．sudo mkdir -p /usr/local/lib/docker/cli-plugins/

２．sudo curl -SL https://github.com/docker/compose/releases/download/v2.36.0/docker-compose-linux-x86_64 -o /usr/local/lib/docker/cli-plugins/docker-compose

３．sudo chmod +x /usr/local/lib/docker/cli-plugins/docker-compose


【必要なファイル】
１．mkdir dockertest
（ディレクトリを作成してcd dockertestで入る）

２．vim compose.yml
（docker compose upで起動）

３．vim nginx/conf.d/default.conf

４．vim Dcokerfile

５．vim public/[ファイル名] → 今回はbbsimagetest.php


【MySQLのテーブルの作成】
１．docker compose exec mysql mysql [DB名] → 今回はexample_db

２．テーブル作成
CREATE TABLE `bbs_entries` (
    `id` INT UNSIGNED NOT NULL AUTO_INCREMENT PRIMARY KEY,
    `body` TEXT NOT NULL,
    `created_at` DATETIME DEFAULT CURRENT_TIMESTAMP
);
